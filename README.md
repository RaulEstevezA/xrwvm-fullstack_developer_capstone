# Full Stack Application Development Project

Educational full-stack car dealership review application based on IBM's
**Full Stack Application Development Project** course on edX.

This repository is a fork/learning implementation of the IBM course project. It
brings together a Django backend, React frontend, MongoDB/Express data service,
and a Flask sentiment-analysis microservice to list dealerships, submit reviews,
and automatically classify review sentiment.

## Project Highlights

- Dealership catalog with state filtering.
- User registration, login, logout, and Django admin access.
- Dealer detail pages with customer reviews.
- Review submission flow for authenticated users.
- Automatic review sentiment marker powered by an NLTK/VADER Flask service.
- MongoDB-backed dealership and review data served through an Express API.
